Slide 2: What is SASS?

What is SASS?
SASS stands for Syntactically Awesome Style Sheets. SASS allows you to develop CSS code dynamically, yielding implicit cross-browser support (depending on library usage).
That sounds great, but I bet it's tricky to set up.
It's pretty similar to other web technologies. There are a few dependencies, but they only come into play during development.
Okay, but you have to configure your website differently to implement SASS, right?
Nope, SASS compiles it's '.scss' files into normal '.css' files as you develop them. You just need to link the new stylesheets to your existing project.
